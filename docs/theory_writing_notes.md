# 理论写作提醒

## 0. 文档目的

本文件用于记录 `Phase 1` 理论核查后，后续论文写作必须注意的文献口径与表述边界。它不是模型规格文档，也不改变 `docs/model_spec.md` 与 `docs/theory_notes.md` 的实现作用。

## 1. 总结结论

| 项目 | 结论 |
|---|---|
| 当前 `Phase 1` 理论框架 | 作为本文 benchmark 的理论基础是成立的，没有原则性错误。 |
| 需要补强之处 | 主要是与经典文献写作口径的贴合度，而不是模型本身有错。 |
| 用途 | 这些提醒主要服务于后续正文写作、文献综述与理论章节表述。 |

## 2. 必须记住的文献口径

### 2.1 Svensson 1997

| 要点 | 写作提醒 |
|---|---|
| 核心概念 | `inflation forecast targeting` |
| 关键区分 | 要区分 `target rules` 与 `instrument rules` |
| 写作禁忌 | 不要把 Svensson 简化成“证明最优政策就是一个 Taylor 型利率规则” |

### 2.2 Clarida, Galí, Gertler 1999

| 要点 | 写作提醒 |
|---|---|
| 文献定位 | canonical New Keynesian 货币政策框架 |
| 核心内容 | 最优政策、规则评价、通胀目标制、承诺与相机抉择 |
| 写作禁忌 | 不要把本文的 reduced-form LQ benchmark 直接说成复现了 Clarida 1999 的完整 NK 结构模型 |

### 2.3 Walsh 2017

| 要点 | 写作提醒 |
|---|---|
| 文献定位 | 货币理论与政策教材中的系统化表述 |
| 关键内容 | 政策目标、最优承诺/相机抉择、targeting regimes、instrument rules、ELB |
| 写作禁忌 | 不要忽略其福利目标与 NK 微观基础背景，只留下一个机械的 LQ 控制框架 |

## 3. 本文理论框架的正确定位

后续写作中，建议始终使用如下表述：

| 正确表述 | 含义 |
|---|---|
| 本文先构造一个可实现的离散时间 LQ benchmark | 它是理论比较基准 |
| 该 benchmark 是对经典最优货币政策问题的可计算化表达 | 不是对全部经典文献结构细节的逐项复现 |
| 在 benchmark 下，最优反馈规则呈线性形式 | 这是本文设定下的结果 |
| RL 在本文中是 Bellman 问题的近似数值求解器 | 这是方法定位 |

## 4. 后续正文必须补上的三个点

| 主题 | 必须补写内容 |
|---|---|
| `target rule` vs `instrument rule` | 在理论综述中明确区分两者 |
| benchmark 的定位 | 明确本文 benchmark 是对经典问题的 reduced-form、可计算化实现 |
| 福利函数来源 | 说明 NK 文献中政策目标常可由家庭效用近似推出，而本文 benchmark 先采用显式二次损失作为规范性目标 |

## 5. 这份提醒的使用方式

| 场景 | 用法 |
|---|---|
| 写文献综述 | 用于避免误述 Svensson / Clarida / Walsh |
| 写理论章节 | 用于说明本文 benchmark 与 canonical NK policy analysis 的关系 |
| 写结论与局限 | 用于强调本文是“可计算化规范分析”，不是完整结构现实世界政策定理 |

---

一句话总结：

当前理论框架没有原则性错误，需要注意的是后续写作时不能把“本文的可计算 benchmark”写成“经典文献本身的完整理论结论”。
