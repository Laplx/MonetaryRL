# Phase 2 数据规格文档

## 0. 文档定位

| 项目 | 说明 |
|---|---|
| 阶段 | `Phase 2` |
| 文档用途 | 冻结本文的数据变量、来源、样本区间、频率、变换方式与数据使用边界。 |
| 参考基线 | 重点参考 `Hinterlang and Tänzer (2021)` 的经验设计。 |
| 最终权威 | `Thesis Proposal.docx`。如与本文件冲突，以开题报告为准。 |

## 1. Phase 2 的任务边界

`Phase 2` 不只是“收集数据”，而是分成三件事：

| 子任务 | 内容 | 输出 |
|---|---|---|
| 2A | 收集并清洗宏观数据 | 可复用的数据集 |
| 2B | 估计环境/转移方程 | 线性 SVAR 环境与非线性 ANN 环境 |
| 2C | 估计经验 Taylor rule | 反事实中的经验政策基准 |

因此，本阶段同时服务两条线：

| 线 | 作用 |
|---|---|
| 环境线 | 估计 $A,B,\Sigma$ 或更一般的转移函数 $f$，供 RL 和反事实模拟使用 |
| 规则线 | 估计经验 Taylor rule 系数，供与 RL/理论规则比较 |

## 2. 方法基线：Hinterlang 的核心做法

本文 `Phase 2` 明确借鉴 `Hinterlang and Tänzer (2021)` 的结构：

| 模块 | Hinterlang 做法 | 本文采用方式 |
|---|---|---|
| 样本 | 美国季度数据 `1987:Q3-2007:Q2` | 作为基准样本 |
| 状态变量 | inflation + output gap | 基准环境采用相同核心变量 |
| 线性环境 | 递归 SVAR | 采用 |
| 非线性环境 | ANN / NARX | 采用，但实现时可不拘泥于 Matlab |
| 反事实对象 | common rules、actual FFR、RL rules | 本文也采用类似比较结构 |

本阶段的重要结论是：

$$
\text{环境估计要做两套：SVAR 与 ANN}
$$

这不是可选项，而是当前最合理的主线设计。

## 3. 数据变量冻结

### 3.1 基准最小变量集

| 变量 | 符号 | 是否必须 | 用途 |
|---|---|---|---|
| 通胀率 | $\pi_t$ | 必须 | 环境状态、政策目标、经验规则 |
| 产出缺口 | $x_t$ | 必须 | 环境状态、政策目标、经验规则 |
| 政策利率 | $i_t$ | 必须 | 环境输入、经验规则、历史政策比较 |

### 3.2 基准口径

默认采用与 Hinterlang 最接近的口径：

| 变量 | 基准口径 | 原因 |
|---|---|---|
| 通胀率 | GDP implicit price deflator 的同比增速 | 与 Hinterlang 一致，也较贴近后续 DSGE/NK 比较语境 |
| 产出缺口 | 实际 GDP 相对潜在 GDP 的百分比偏离 | 与 Hinterlang 一致，可解释性强 |
| 政策利率 | Effective Federal Funds Rate | 与文献和历史政策比较一致 |

### 3.3 可选稳健性口径

| 变量 | 可选替代 | 用途 |
|---|---|---|
| 通胀率 | PCE inflation / core PCE inflation | 稳健性分析 |
| 产出缺口 | HP filter gap / CBO output gap 直接序列 | 稳健性分析 |
| 政策利率 | target rate 或 shadow rate | 特殊扩展，不作为主线 |

## 4. 官方来源与下载优先级

### 4.1 主来源

优先使用 FRED 汇总的官方美国宏观数据，原因是稳定、统一、便于复现。

| 变量 | FRED 代码 | 官方来源 | 链接 |
|---|---|---|---|
| GDP deflator | `GDPDEF` | BEA via FRED | https://fred.stlouisfed.org/series/GDPDEF |
| Real GDP | `GDPC1` | BEA via FRED | https://fred.stlouisfed.org/series/GDPC1 |
| Potential Real GDP | `GDPPOT` | CBO via FRED | https://fred.stlouisfed.org/series/GDPPOT |
| Effective Fed Funds Rate | `FEDFUNDS` | Board of Governors via FRED | https://fred.stlouisfed.org/series/FEDFUNDS |

### 4.2 备选来源

| 机构 | 用途 | 说明 |
|---|---|---|
| BEA | GDP 与 deflator 原始来源 | 若 FRED 聚合口径不满足需要，可回到 BEA |
| CBO | 潜在产出原始来源 | 若需要核对 GDPPOT 背后定义 |
| FOMC/FRB | 政策利率与政策框架文本 | 用于写作，不是主数据源 |

## 5. 样本区间冻结

### 5.1 基准估计样本

环境估计与经验规则估计的主样本冻结为：

$$
1987{:}Q3 \text{ 至 } 2007{:}Q2
$$

理由如下：

| 原因 | 说明 |
|---|---|
| 与 Hinterlang 对齐 | 便于直接文献比较 |
| 避开金融危机后的非常规政策时期 | 避免把 QE 等额外政策工具混入单一利率规则环境 |
| Greenspan 起点 | 与“隐性通胀目标制”时期更一致 |

### 5.2 扩展展示样本

可在后续静态处方图或补充图中延伸到危机后时期，但原则如下：

| 场景 | 是否允许 |
|---|---|
| 用更长样本展示规则处方差异 | 允许 |
| 用危机后样本重新训练主环境 | 暂不作为主线 |
| 用危机后样本做稳健性扩展 | 允许，但留到后续阶段 |

## 6. 变量变换规则冻结

### 6.1 通胀率

默认定义为 GDP deflator 的同比增速：

$$
\pi_t = 100 \times \left(\log(P_t)-\log(P_{t-4})\right)
$$

或近似等价地用同比百分比变化率。

说明：

| 事项 | 规则 |
|---|---|
| 主文稿 | 统一写为同比通胀率 |
| 代码实现 | 推荐统一用对数差近似百分比 |
| 若直接使用 FRED 百分比变换系列 | 必须记录 series code 与变换来源 |

### 6.2 产出缺口

默认定义为实际 GDP 相对潜在 GDP 的百分比偏离：

$$
x_t = 100 \times \left(\frac{Y_t - Y_t^{pot}}{Y_t^{pot}}\right)
$$

或用对数近似

$$
x_t \approx 100 \times \left(\log Y_t - \log Y_t^{pot}\right)
$$

主线建议采用后一种对数近似，便于与宏观模型表达一致。

### 6.3 政策利率

`FEDFUNDS` 为月度频率，因此需聚合到季度频率。主线规则冻结为：

$$
i_t = \text{季度内月度有效联邦基金利率的平均值}
$$

### 6.4 中心化与标准化

| 处理 | 何时使用 | 说明 |
|---|---|---|
| 中心化 | 建模与理论分析 | 如 $\tilde{\pi}_t=\pi_t-\pi^\ast$，$\tilde{i}_t=i_t-i^\ast$ |
| 标准化 | RL 训练 | 只用于数值稳定，不改变经济学定义 |

## 7. 数据集组织规则

建议形成三个层次的数据表：

| 文件 | 内容 | 用途 |
|---|---|---|
| `data/raw/` | 原始下载数据 | 保留原始来源 |
| `data/processed/macro_quarterly.csv` | 清洗后的季度数据 | 环境估计、Taylor 回归 |
| `data/processed/model_inputs.csv` | 中心化/标准化后建模输入 | RL 与 benchmark 实现 |

清洗后数据至少包含：

| 字段 | 含义 |
|---|---|
| `date` | 季度时间索引 |
| `inflation` | 通胀率 |
| `output_gap` | 产出缺口 |
| `policy_rate` | 政策利率 |
| `inflation_gap` | 通胀偏离目标 |
| `policy_rate_gap` | 利率偏离稳态/中性水平 |

## 8. 数据质量与解释边界

### 8.1 必须记录的边界

| 问题 | 说明 |
|---|---|
| 修订数据问题 | FRED/BEA/CBO 提供的是 ex post revised data，不是实时数据 |
| 实时数据偏误 | 经验 Taylor 回归若使用修订数据，不能直接解释为决策时真实可观测信息 |
| 非常规政策时期 | 危机后样本中利率受 ELB 与其他工具共同影响，解释更复杂 |

### 8.2 本文处理原则

| 事项 | 规则 |
|---|---|
| 主样本 | 使用修订后的完整数据集 |
| 正文说明 | 必须注明不是实时数据分析 |
| 稳健性 | 如有时间，可增加实时数据或替代变量讨论 |

## 9. 对 Phase 2B 的直接约束

环境估计要严格区分两条线：

| 环境 | 形式 | 用途 |
|---|---|---|
| 线性环境 | 递归 SVAR | benchmark 近似现实环境、与 Hinterlang 对照 |
| 非线性环境 | ANN / NARX 风格监督学习 | 非线性反事实环境、RL 扩展主环境 |

二者都基于同一清洗后季度数据构造，不允许混用不同口径。

## 10. 对 Phase 2C 的直接约束

经验 Taylor rule 的估计样本、变量口径必须与环境估计共用同一份季度数据，否则反事实比较不干净。

建议的基准经验规则：

$$
i_t = \alpha + \phi_\pi \pi_t + \phi_x x_t + \phi_i i_{t-1} + u_t
$$

若后续需要，也可增加不含平滑项的简单规格作为附表。

## 11. 当前已知的潜在困难

| 类型 | 当前判断 | 处理方式 |
|---|---|---|
| 数据识别 | 难度低 | 主要 series 已明确 |
| 数据下载到本地 | 可能受当前环境网络限制 | 若自动下载失败，需要用户手动下载 CSV |
| 文献参数表获取 | 中等 | 大部分可自主搜索；教材或付费文献可能需要用户补 PDF |
| 潜在产出定义一致性 | 中等 | 默认使用 `GDPPOT`，若后续发现口径不一致再调整 |

## 12. Phase 2 数据部分验收标准

| 验收项 | 要求 |
|---|---|
| 变量集合明确 | inflation、output gap、policy rate 已定义 |
| 来源明确 | 主来源与备选来源已列出 |
| 口径明确 | 同比、季度平均、对数差等变换已固定 |
| 样本明确 | 主样本 `1987:Q3-2007:Q2` |
| 环境/规则两条线已区分 | 已完成 |

---

一句话总结：

`Phase 2` 的数据主线是：使用与 Hinterlang 尽量一致的美国季度宏观数据，在同一份样本上同时支撑两套环境估计（SVAR 与 ANN）和一条经验 Taylor rule 估计线。
