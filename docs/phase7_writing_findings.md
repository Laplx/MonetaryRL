# Phase 7 可直接写入论文的观察与结论

## 0. 文档定位

本文件记录截至当前版本最值得写入论文正文或结果讨论部分的观察。这里强调的是“当前实验结果支持的结论”，不是最终不可更改的定论。后续若进一步加大训练预算、增加种子或调整环境强度，应同步更新。

## 1. 当前实验矩阵

本轮主结果采用如下结构：

| 维度 | 内容 |
|---|---|
| baseline | `1` 个 benchmark 环境 |
| 扭曲环境 | `3` 档 nonlinear Phillips、`3` 档 ZLB/ELB-tightness、`3` 档 asymmetric loss |
| RL 算法 | `PPO`、`TD3`、`SAC` |
| 随机种子 | `7`、`29`、`43` |

因此，RL 主矩阵为：

$$
(1 + 3 + 3 + 3) \times 3
$$

并且每个环境还都保留了外部对照规则：

| 对照规则 | 说明 |
|---|---|
| Riccati reference | benchmark 理论线性最优规则，在扩展环境中视为外推参考 |
| linear policy search | 更贴近当前低维问题结构的强基线 |
| empirical Taylor rule | Phase 2 估计得到的外部经验规则 |
| zero policy | 极弱基线 |

## 2. 最值得强调的总体结论

### 2.1 算法相对表现具有明显环境依赖性

最重要的结论不是“某个算法全局最好”，而是：

| 环境类型 | 当前最强 RL | 观察 |
|---|---|---|
| benchmark | `SAC`（当前多 seed / 当前预算下） | `PPO` 与 `TD3` 也可运行，但相对更弱 |
| nonlinear Phillips | `SAC` | 非线性强度上升后，`PPO` 相对掉队更明显 |
| ZLB/ELB-tightness | `TD3` 或 `SAC` | `PPO` 在约束强化时明显恶化 |
| asymmetric loss | `SAC` | 相比 `PPO` 和 `TD3` 更稳 |

可写成论文中的一个核心判断：

> 在本文的扩展环境中，RL 算法排序并不稳定，而是随环境结构变化而变化。这说明算法表现具有显著的环境依赖性。

### 2.2 PPO 不是所有环境下最合适的算法

这是当前最清晰、也最重要的负结果之一。

| 环境 | 当前判断 |
|---|---|
| benchmark | PPO 可用，但在当前多 seed 固定预算下不如 SAC |
| nonlinear | PPO 显著落后于 SAC，且随着非线性增强差距扩大 |
| ZLB | PPO 表现最弱，尤其在 stronger tier 下恶化明显 |
| asymmetric | PPO 一直弱于 SAC 和 TD3 |

可写作：

> PPO 作为主算法具有实现稳定和解释便利的优点，但并不构成所有环境下的最优选择。尤其在更强约束和更强扭曲环境中，off-policy 连续控制算法更有优势。

### 2.3 线性参考规则的鲁棒性比预期更强

这一点也很值得写。

| 现象 | 含义 |
|---|---|
| 在三档 nonlinear 环境中，Riccati reference 一直强于所有 RL 算法 | 当前 nonlinear 扭曲虽然有效，但尚未足以让线性参考规则失效 |
| 在 asymmetric 环境中，Riccati reference 与 linear policy search 依然非常强 | 说明当前 asymmetric 扭曲更多改变了福利权重，而不是强烈改变最优反馈结构 |

这意味着：

> 扩展环境虽然引入了非线性、约束和非对称目标，但最优政策的主要结构在相当范围内仍接近线性反馈。RL 的优势更多体现在适应特定扭曲，而非完全颠覆线性规则。

## 3. 分模块可写结论

### 3.1 benchmark

当前可写结论：

| 结论 | 说明 |
|---|---|
| benchmark 仍以 Riccati 理论解为唯一规范基准 | 不变 |
| RL 在 benchmark 中能够学到有效规则 | 成立 |
| 但在当前多 seed 固定预算下，SAC 比 PPO 更接近 benchmark reference | 当前结果支持 |
| linear policy search 是非常强的 benchmark 专用基线 | 应在正文强调 |

当前不宜写：

| 不宜写的说法 | 原因 |
|---|---|
| PPO 已稳定逼近理论最优 | 当前多 seed 结果不支持 |
| PPO 是 benchmark 中表现最好的 RL 算法 | 当前结果不支持 |

### 3.2 nonlinear Phillips

当前可写结论：

| 结论 | 说明 |
|---|---|
| nonlinear Phillips 扭曲强度上升时，PPO 相对劣化显著 | 结果支持 |
| SAC 在三档 nonlinear 环境中都保持最强 RL 表现 | 结果支持 |
| benchmark 的线性 Riccati rule 在当前 nonlinear 校准下仍然非常强 | 结果支持 |

当前最值得讨论的点：

> 当前 nonlinear 环境尚未使线性参考规则失效，这说明本文引入的非线性属于“有经济意义但仍有限度”的扭曲，而不是彻底改变最优反馈结构的极端非线性。

### 3.3 ZLB / ELB-tightness

当前可写结论：

| 结论 | 说明 |
|---|---|
| 约束越强，PPO 的损失恶化越快 | 很明显 |
| TD3 在 stronger bound 下优于 SAC 和 PPO | 结果支持 |
| SAC 与 TD3 都明显优于 PPO | 结果支持 |

这部分很适合写成：

> 与 benchmark 和 mild nonlinear 环境不同，约束强化环境更偏向于 off-policy 连续控制算法。这说明约束型货币政策优化问题的算法难点与无约束环境并不相同。

### 3.4 asymmetric loss

当前可写结论：

| 结论 | 说明 |
|---|---|
| asymmetric 扭曲增强后，所有算法损失都上升 | 结果支持 |
| SAC 在三档 asymmetric 环境中都最稳且最强 | 结果支持 |
| PPO 在 asymmetric 环境下持续明显落后 | 结果支持 |

这部分适合强调：

> 当中央银行对上行通胀和下行产出缺口赋予额外惩罚时，策略学习问题对算法稳定性提出了更高要求。当前结果显示，SAC 对此更有适应性。

## 4. 当前最值得写成论文亮点的三句话

建议优先保留下面三句风格的结论：

1. RL 算法的优劣排序并不固定，而是随货币政策环境中的非线性、约束和目标扭曲而系统变化。
2. 在 tighter ZLB/ELB 环境中，TD3 和 SAC 明显优于 PPO，说明约束型连续控制问题与无约束 benchmark 存在不同的算法适配性。
3. 即使在引入 nonlinear Phillips 和 asymmetric loss 后，线性参考规则仍表现出较强鲁棒性，表明当前扩展环境尚未完全改变最优反馈结构的基本形态。

## 5. 当前需要保守措辞的地方

| 问题 | 保守措辞 |
|---|---|
| 当前 PPO 与之前单次 tuned PPO 结果不一致 | 应明确区分“单次强化结果”和“多 seed 固定预算稳健性结果” |
| ZLB 三档严格说并不全是纯粹的零利率下限 | 应写作“ZLB/ELB-tightness tiers” 或 “逐步收紧的有效下界环境” |
| nonlinear 外推规则仍然最强 | 不应解释为“RL 在 nonlinear 环境中无价值”，而应解释为“当前非线性强度尚未完全压倒线性参考结构” |
