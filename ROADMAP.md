## 1. 总目标与研究问题

| 层面 | 内容 |
|---|---|
| 总目标 | 在一个结构清晰、参数具现实量级的随机宏观货币政策模型中，系统比较经典最优控制方法与强化学习方法在央行利率规则求解上的表现。 |
| 基准问题 | 在可解析的线性二次型（LQ）情形下，RL 学到的政策是否接近理论最优政策？ |
| 扩展问题 | 在不可解析或难解析情形下，如非线性 Phillips 曲线、零利率下界（ZLB）、可能的非对称损失等，RL 是否仍能稳定求得表现良好的反馈规则？ |
| 经验问题 | 用现实参数和历史数据校准后，RL 规则、理论规则、经验 Taylor rule、历史实际政策在反事实模拟中的福利损失如何比较？ |
| 贡献目标 | 不仅比较性能，更要比较政策函数结构、经济含义、稳定性与可解释性。 |

## 2. 研究边界与执行原则

| 主题 | 执行原则 |
|---|---|
| 模型边界 | 先从低维、可解释、可复现实验的模型出发，再扩展到非线性或约束情形。避免一开始进入高维 DSGE 大系统。 |

| 文献边界 | `Hinterlang`、`Charpentier`、`Chen Mingli` 是关键参照，但不改变开题报告的主线定位。 |
| 结果边界 | 不能只报告 loss 最优值，必须报告政策函数形状、状态响应、训练稳定性与经济学解释。 |
| 写作边界 | 正文中必须明确说明 Lucas critique 风险，即当政策规则改变时，转移方程可能不再不变。 |
| 复现边界 | 所有实验必须可重复，固定随机种子，保存配置、模型参数、输出图表与日志。 |

## 5. Phase 0：研究框架冻结

### 5.1 目标

在动手写代码前，先把开题报告中模糊处固定为可执行的研究设计，避免后期频繁改模型。

### 5.2 必须确定的问题

| 主题 | 需要确定的内容 | 建议 |
|---|---|---|
| 时间框架 | 离散时间还是连续时间 | 建议先做离散时间，便于数据、Riccati 与 RL 环境统一。 |
| 状态变量 | 最少包含哪些状态 | 至少包含通胀 $\pi_t$、产出缺口 $x_t$；若有利率平滑，需加入上一期利率 $i_{t-1}$。 |
| 控制变量 | 央行控制项 | 名义利率 $i_t$。 |
| 冲击过程 | 随机性如何进入 | 建议先用离散高斯冲击；如后续需要，可再写成连续时间近似。 |
| 目标函数 | 央行损失如何定义 | 建议基准为二次损失，扩展中再引入非对称项或约束。 |
| 评价指标 | 结果如何比较 | 折现总损失、无条件均值/方差、状态轨迹、政策函数、训练稳定性。 |
| 策略比较对象 | 与谁比较 | 理论最优规则、经验 Taylor rule、历史实际政策、可能的简单规则。 |

### 5.3 建议的基准状态表示

建议用增广状态

$$
s_t = \begin{bmatrix}
\pi_t \\
x_t \\
i_{t-1}
\end{bmatrix}
$$

理由如下：

| 原因 | 说明 |
|---|---|
| 利率平滑 | 若损失函数包含 $(i_t - i_{t-1})^2$，则必须把 $i_{t-1}$ 作为状态。 |
| LQ 一致性 | 增广状态后仍可保持线性状态转移与二次损失结构。 |
| RL 一致性 | RL 环境输入可以与经典最优控制完全对齐，减少比较偏差。 |

## 6. Phase 1：理论回顾与推导

这一阶段必须形成可直接写进论文正文或附录的理论内容。

### 6.1 最优货币政策的 LQ 框架

#### 6.1.1 状态转移

基准可写为

$$
s_{t+1} = A s_t + B i_t + \Sigma \varepsilon_{t+1}, \quad \varepsilon_{t+1} \sim \mathcal{N}(0, I)
$$

其中 $s_t$ 为增广状态，$i_t$ 为名义利率。

#### 6.1.2 央行损失函数

建议基准损失为

$$
\ell(s_t, i_t) = \lambda_\pi (\pi_t - \pi^\ast)^2 + \lambda_x x_t^2 + \lambda_i (i_t - i_{t-1})^2
$$

若通胀目标在数据处理中已中心化到 $0$，则可写为

$$
\ell(s_t, i_t) = \lambda_\pi \pi_t^2 + \lambda_x x_t^2 + \lambda_i (i_t - i_{t-1})^2
$$

长期目标为最小化贴现损失

$$
V(s_t) = \min_{\{i_{t+k}\}_{k \ge 0}} \mathbb{E}_t \left[\sum_{k=0}^{\infty} \beta^k \ell(s_{t+k}, i_{t+k}) \right]
$$

#### 6.1.3 Bellman 方程

离散时间最优控制可写为

$$
V(s) = \min_i \left\{ \ell(s, i) + \beta \mathbb{E}\left[V(s') \mid s, i\right] \right\}
$$

在线性状态转移与二次损失下，值函数猜测为二次型

$$
V(s) = s^\top P s + c
$$

进而得到离散代数 Riccati 方程。最优反馈规则为线性形式

$$
i_t = -F s_t
$$

其中

$$
F = \left(R + \beta B^\top P B\right)^{-1} \beta B^\top P A
$$

这里的 $R$ 表示控制项对应的二次惩罚矩阵，$Q$ 表示状态项对应的二次惩罚矩阵，$P$ 满足 Riccati 方程

$$
P = Q + \beta A^\top P A - \beta^2 A^\top P B \left(R + \beta B^\top P B\right)^{-1} B^\top P A
$$

若增广状态包含 $i_{t-1}$，则上式依旧成立，只是 $A,B,Q,R$ 的维度相应变化。

#### 6.1.4 与 Taylor rule 的关系

如果最优反馈规则写成

$$
i_t = \phi_\pi \pi_t + \phi_x x_t + \phi_i i_{t-1}
$$

则它与经验 Taylor rule 在形式上相近，但含义不同：

| 项目 | 最优反馈规则 | 经验 Taylor rule |
|---|---|---|
| 来源 | 优化问题的解 | 回归估计或规则设定 |
| 系数 | 由模型、损失函数、冲击结构共同决定 | 由样本数据或先验政策规则决定 |
| 作用 | 规范分析基准 | 实证比较基准 |

### 6.2 非线性与约束情形

开题报告要求不能停留在 LQ 情形，必须进入不可解析或难解析情形。

#### 6.2.1 非线性 Phillips 曲线

可考虑用如下形式表示凸性或非线性响应：

$$
\pi_{t+1} = a_\pi \pi_t + a_x x_t + a_i i_t + a_{xx} x_t^2 + a_{\pi x} \pi_t x_t + \sigma_\pi \varepsilon^\pi_{t+1}
$$

或更一般地写为

$$
\pi_{t+1} = f_\pi(\pi_t, x_t, i_t) + \sigma_\pi \varepsilon^\pi_{t+1}
$$

### 6.2.2 非线性 IS 或状态转移

$$
x_{t+1} = f_x(\pi_t, x_t, i_t, i_{t-1}) + \sigma_x \varepsilon^x_{t+1}
$$

一旦 $f_\pi, f_x$ 非线性，解析解一般消失，传统数值方法可能依赖离散网格并受维数灾难限制。

#### 6.2.3 ZLB 约束

零利率下界可写为

$$
i_t \ge 0
$$

或者在实现层面写成

$$
i_t = \max(0, \tilde{i}_t)
$$

其中 $\tilde{i}_t$ 是无约束策略网络或解析规则的原始输出。

#### 6.2.4 非对称损失

若要体现央行对高通胀和低通胀、正产出缺口和负产出缺口的非对称偏好，可考虑

$$
\ell_t = \lambda_\pi \pi_t^2 + \lambda_x x_t^2 + \lambda_i (i_t - i_{t-1})^2 + \lambda_{\pi,+}\max(\pi_t,0)^2 + \lambda_{x,-}\max(-x_t,0)^2
$$

此项可作为可选扩展，不建议在最初版本引入过多复杂度。

### 6.3 强化学习作为数值最优控制

#### 6.3.1 MDP 表述

把央行问题写成马尔可夫决策过程：

| 元素 | 定义 |
|---|---|
| 状态 $s_t$ | 包含 $\pi_t, x_t, i_{t-1}$ 的向量 |
| 动作 $a_t$ | 当期政策利率 $i_t$ |
| 转移 | 由宏观模型给出，即 $p(s_{t+1}\mid s_t, a_t)$ |
| 即时回报 | $r_t = -\ell(s_t, a_t)$ |
| 目标 | 最大化折现回报和，即最小化折现损失和 |

策略目标函数为

$$
J(\pi_\theta) = \mathbb{E}\left[\sum_{t=0}^{\infty} \gamma^t r_t \right]
$$

由于 $r_t = -\ell_t$，因此等价于最小化央行损失。

#### 6.3.2 为什么 RL 可以用于本文

| 理由 | 说明 |
|---|---|
| 与 Bellman 一致 | RL 本质上是在做 Bellman 方程的近似求解。 |
| 不显式解 HJB | 可在复杂状态空间中直接学习反馈策略。 |
| 适合约束与非线性 | ZLB、非线性状态转移、复杂损失函数都容易写进环境。 |
| 可直接比较 | 在 LQ benchmark 下可与理论最优反馈规则逐项对照。 |

#### 6.3.3 算法选择建议

| 算法 | 是否推荐 | 原因 |
|---|---|---|
| PPO | 推荐作为首选 | 稳定、实现成熟、适合连续动作、调参经验丰富。 |
| SAC | 推荐作为备选 | 连续动作表现强，样本效率较高。 |
| DDPG | 可参考但不优先 | 在很多任务上较脆弱，`Hinterlang` 使用它主要是其时点较早。 |
| A2C/A3C | 可作辅助基线 | 结构简单，但未必是连续控制首选。 |

建议路线：先用 PPO 建立稳定基线，再考虑 SAC；不建议一开始同时开多个算法线。

### 6.4 反事实模拟与福利分析

这一部分是正文必须出现的理论与方法内容。

#### 6.4.1 反事实路径生成

给定初始状态与冲击序列，对不同政策规则分别生成

$$
\{ \pi_t, x_t, i_t \}_{t=0}^{T}
$$

并比较各自的损失、均值、波动与极端状态行为。

#### 6.4.2 福利损失比较

建议至少报告三种量：

| 指标 | 解释 |
|---|---|
| 折现总损失 | 核心 welfare 指标 |
| 平均单期损失 | 便于跨实验比较 |
| 状态波动指标 | 如 $\mathrm{Var}(\pi_t)$、$\mathrm{Var}(x_t)$、$\mathrm{Var}(\Delta i_t)$ |

#### 6.4.3 结构解释

不仅比较谁的 loss 更低，还要解释：

| 问题 | 要回答的内容 |
|---|---|
| 政策函数形状 | RL 规则是否近似线性，是否在某些区域更强烈或更保守？ |
| 非线性来源 | ZLB 或非线性 Phillips 曲线如何改变最优反应？ |
| 经济含义 | 在高通胀、深度负缺口等区域，RL 是否表现出更强状态依赖性？ |

### 6.5 理论阶段退出标准

| 标准 | 要求 |
|---|---|
| 基准模型 | 已写出状态转移、损失函数、Bellman 方程、Riccati 公式。 |
| 扩展模型 | 已写出至少一种非线性扩展和一种约束扩展。 |

## 8. Phase 3：基准 LQ 模型实现

### 8.1 目标

得到一个最小可运行、可解析、可与 RL 逐项比较的 benchmark。

### 8.2 建议的最小模型

可采用线性状态转移

$$
s_{t+1} = A s_t + B i_t + \Sigma \varepsilon_{t+1}
$$

其中

$$
s_t = \begin{bmatrix} \pi_t & x_t & i_{t-1} \end{bmatrix}^\top
$$

损失函数为

$$
\ell_t = \lambda_\pi \pi_t^2 + \lambda_x x_t^2 + \lambda_i (i_t - i_{t-1})^2
$$

### 8.3 关键实现要求

| 要求 | 说明 |
|---|---|
| 参数可替换 | 所有参数必须由配置文件读入，不可硬编码。 |
| 随机种子固定 | 方便与 RL 使用同一冲击序列对照。 |
| 状态更新可测试 | 必须能对单步转移写单元测试。 |
| 目标函数一致 | 经典方法与 RL 使用完全相同的单期损失定义。 |

### 8.4 退出标准

| 标准 | 要求 |
|---|---|
| 模型可运行 | 可给定初值与冲击生成时间序列。 |
| 参数透明 | 可打印或导出参数表。 |
| 文档齐全 | 已说明每个状态与参数的经济含义。 |

## 9. Phase 4：经典最优控制求解

### 9.1 目标

在 benchmark 下得到理论最优政策规则，作为 RL 的真值基准。

### 9.2 实现路线

| 步骤 | 内容 |
|---|---|
| Step 1 | 写出 $Q,R,A,B$ 矩阵。 |
| Step 2 | 求解离散 Riccati 方程，得到 $P$。 |
| Step 3 | 求解最优反馈矩阵 $F$。 |
| Step 4 | 在相同初值与冲击下生成最优轨迹。 |
| Step 5 | 导出政策系数、状态路径与折现总损失。 |

### 9.3 必做检查

| 检查项 | 说明 |
|---|---|
| 稳定性 | 闭环系统特征值是否在单位圆内。 |
| 合理性 | 系数符号是否符合直觉，如对高通胀应提高利率。 |
| 数值一致性 | 与独立库函数或理论公式交叉验证。 |

### 9.4 输出物

| 输出 | 说明 |
|---|---|
| 最优反馈矩阵 | 线性规则系数。 |
| 基准路径图 | $\pi_t,x_t,i_t$ 动态路径。 |
| 福利表 | 折现总损失与分项损失。 |
| 理论说明 | 可直接写进论文理论部分。 |

## 10. Phase 5：RL 环境与基线训练

### 10.1 目标

在与 benchmark 完全一致的环境中训练 RL，使其成为严格可比的数值求解器。

### 10.2 环境定义

| 元素 | 要求 |
|---|---|
| 观察空间 | 默认与状态 $s_t$ 一致。 |
| 动作空间 | 连续利率区间；若有 ZLB，可设下界为 $0$。 |
| 奖励函数 | 必须定义为 $r_t=-\ell_t$。 |
| episode 设计 | 固定长度或无穷期截断近似，长度应足够反映长期损失。 |
| 终止条件 | 除固定长度外，可设置状态爆炸保护。 |

### 10.3 训练策略

| 项目 | 建议 |
|---|---|
| 首选算法 | PPO |
| 备选算法 | SAC |
| 输入处理 | 对状态做标准化，对动作做合理缩放。 |
| 多种种子 | 至少 5 个随机种子，防止偶然成功。 |
| 检查点 | 训练中定期保存模型与评估结果。 |

### 10.4 必做诊断

| 诊断项 | 说明 |
|---|---|
| 学习曲线 | 平均回报是否稳定上升。 |
| 策略输出范围 | 是否频繁卡在动作边界。 |
| 状态分布 | 是否出现极端爆炸路径。 |
| 重复性 | 不同种子结果是否一致。 |

## 11. Phase 6：Benchmark 对照实验

### 11.1 目标

严格比较理论最优控制与 RL 在可解析 benchmark 下的接近程度。

### 11.2 核心比较维度

| 维度 | 比较方式 |
|---|---|
| 政策函数 | 比较 RL 输出与线性最优规则的拟合程度。 |
| 总损失 | 比较折现总损失差距。 |
| 单期路径 | 比较相同冲击下的路径偏离。 |
| 局部导数 | 估计 RL 对 $\pi_t,x_t,i_{t-1}$ 的边际反应。 |
| 训练稳定性 | 比较不同种子下结果方差。 |

### 11.3 必做图表

| 图表 | 内容 |
|---|---|
| 图 1 | 训练回报曲线 |
| 图 2 | 理论最优规则与 RL 规则的散点拟合图 |
| 图 3 | 相同冲击下的状态路径对比图 |
| 表 1 | 总损失、平均损失、状态方差比较 |
| 表 2 | RL 拟合出的近似线性系数与理论系数对比 |

### 11.4 判定逻辑

若 RL 在 benchmark 下无法逼近理论最优，则不能进入扩展模型，必须先回头修正环境、奖励、算法或归一化设置。

## 12. Phase 7：非线性与约束扩展

### 12.1 目标

展示 RL 在传统解析方法受限时的价值，这是论文创新的重要部分。

### 12.2 建议扩展顺序

| 顺序 | 扩展 | 说明 |
|---|---|---|
| 1 | 非线性 Phillips 曲线 | 最接近开题报告表述，优先实现。 |
| 2 | ZLB 约束 | 政策含义强，文献联系清晰。 |
| 3 | 非对称损失 | 如时间允许再加入。 |

### 12.3 扩展实验设计

| 实验 | 需要比较的对象 |
|---|---|
| 非线性环境 + 无约束 | RL 与线性规则外推结果比较 |
| 线性环境 + ZLB | RL 与受限简单规则比较 |
| 非线性环境 + ZLB | 重点展示 RL 的状态依赖策略 |

### 12.4 必须输出的内容

| 输出 | 内容 |
|---|---|
| 政策函数热图 | 展示 $i_t$ 对 $(\pi_t,x_t)$ 的非线性反应。 |
| 区域分析 | 在高通胀、低通胀、深度衰退区域分别分析策略差异。 |
| 稳定性报告 | 训练是否更难、是否更依赖超参数。 |

## 13. Phase 8：经验 Taylor rule 与历史反事实

### 13.1 目标

将模型结果与现实政策讨论连接起来，形成毕业论文中最具经济学解释力的一部分。

### 13.2 经验规则估计

建议估计

$$
i_t = \alpha + \phi_\pi \pi_t + \phi_x x_t + \phi_i i_{t-1} + u_t
$$

必要时可考虑前瞻项或不同滞后规格，但初版不宜过多扩展。

### 13.3 反事实对象

| 对象 | 说明 |
|---|---|
| 理论最优规则 | benchmark 规范基准 |
| RL 规则 | 数值最优控制候选 |
| 经验 Taylor rule | 样本估计规则 |
| 历史实际政策 | 数据中的真实利率路径 |

### 13.4 比较指标

| 指标 | 说明 |
|---|---|
| 折现总损失 | 核心福利比较 |
| 平均通胀偏离 | 离目标有多远 |
| 平均产出缺口偏离 | 稳定产出效果 |
| 利率平滑程度 | 是否导致过度波动 |
| 极端时期表现 | 危机或高通胀阶段是否更稳健 |

### 13.5 重要写作提醒

在这一部分必须明确说明：若转移方程由历史数据估计得到，则在反事实中默认政策改变不反馈到结构本身，这是一种方法近似，存在 Lucas critique 风险。

## 14. Phase 9：稳健性、消融与解释

### 14.1 目标

证明结果不是由偶然参数、单次种子或个别超参数驱动。

### 14.2 建议稳健性项

| 类型 | 具体内容 |
|---|---|
| 参数稳健性 | 改变 $\lambda_\pi,\lambda_x,\lambda_i,\beta$ 等。 |
| 冲击稳健性 | 改变冲击方差与相关性。 |
| 算法稳健性 | PPO 与 SAC 对比。 |
| 规格稳健性 | 是否加入利率滞后、是否中心化状态变量。 |
| 数据稳健性 | 不同样本区间、不同通胀定义。 |

### 14.3 消融实验

| 消融项 | 目的 |
|---|---|
| 去掉利率平滑项 | 看政策是否变得更激进。 |
| 去掉非线性项 | 验证性能提升是否来自捕捉非线性。 |
| 不做状态标准化 | 观察训练稳定性是否下降。 |

### 14.4 解释任务

必须把 RL 规则尽量翻译回经济学语言：

| 任务 | 说明 |
|---|---|
| 线性近似回归 | 用回归逼近训练好的策略网络，观察主要反应系数。 |
| 局部斜率 | 在不同状态区域估计局部政策敏感度。 |
| 区域切片图 | 固定一个状态变量，对另一个变量画策略切片。 |

## 15. Phase 10：论文写作与组装

### 15.1 建议章节结构

| 章节 | 内容 |
|---|---|
| 第 1 章 绪论 | 研究背景、问题、贡献、结构安排。 |
| 第 2 章 文献综述与研究定位 | 基于 `Literature Review.pdf`，但按 L1/L2/L3 重新组织并压实与本文关系。 |
| 第 3 章 理论框架与方法 | LQ 最优控制、Bellman/Riccati、RL-MDP 对应、扩展模型设定。 |
| 第 4 章 数据、校准与实验设计 | 数据来源、参数、样本、经验规则估计、训练配置。 |
| 第 5 章 Benchmark 结果 | 理论最优与 RL 的 benchmark 对照。 |
| 第 6 章 非线性与约束扩展结果 | 非线性 Phillips 曲线、ZLB、可选非对称损失。 |
| 第 7 章 反事实与稳健性分析 | 历史政策、Taylor rule、福利比较与稳健性。 |

### 15.2 写作顺序建议

| 4 | 扩展实验完成后写第 6、7 章。 |

### 15.3 图表管理规则

| 规则 | 要求 |

| 一图一义 | 每张图只讲一个问题，避免信息过载。 |

## 18. 风险清单

| 风险 | 表现 | 应对 |
|---|---|---|
| RL 不收敛 | 回报剧烈波动、策略卡边界 | 简化环境、标准化状态、先回到线性 benchmark。 |
| Benchmark 不一致 | RL 与理论规则差异异常大 | 检查奖励函数、状态定义、episode 截断与折现因子。 |
| 非线性扩展过早 | 代码复杂度急剧上升 | 坚持先做 E1/E2。 |
| 经验转移偏误 | 反事实结果看似很强但解释薄弱 | 明确写出 Lucas critique 与 reduced-form 限制。 |
| 写作滞后 | 实验做完但正文结构散乱 | 每阶段同步沉淀可直接贴进正文的材料。 |
