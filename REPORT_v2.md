# 货币政策强化学习项目中期报告（v2）

本报告以 `Thesis Proposal.docx` 为最终口径，结合当前项目主线结果，对已完成工作、阶段性观察与可支持结论做一次收束性汇报。这里有意不再重复早期零散实验，也不采用 `phase7` 之前不完整的局部结果；涉及扩展环境时，统一以 `outputs/phase7/matrix/` 的完整矩阵为准。对 `Hinterlang and Tänzer (2021)` 的借鉴主要体现在“经验环境 + 反事实比较 + 规则可视化解释”的研究思路上，但本文当前结论更为审慎，重点仍放在理论 benchmark、环境扩展与经验迁移边界的系统比较。

## 一、目前已经完成的主要工作

首先，项目已经建立起一条较清晰的比较链条。理论部分完成了线性—二次 benchmark 的求解，并得到 Riccati 参考规则，因而强化学习结果不再是孤立展示，而是可以和规范意义上的最优反馈规则直接比较。随后，项目在 benchmark 环境中补齐了更强的对照组：除 PPO 外，还加入了 linear policy search、TD3 和 SAC，使 benchmark 不再只是“RL 能否跑通”的验证，而变成“RL 与强线性基线、理论最优解之间相差多少”的规范比较。

其次，扩展环境已经形成完整矩阵。当前 `phase7` 主结果不再局限于单一 nonlinear 或单一 ZLB 设定，而是覆盖了三类扭曲：非线性 Phillips 曲线、逐步收紧的有效下界环境、非对称损失环境，并统一使用多 seed、固定预算的比较协议。这样做的意义在于，项目现在不仅能回答“RL 是否有效”，还能回答“哪类算法在哪种政策环境下更适合”。

再次，经验环境的第一轮连接已经完成。`phase8` 已将历史实际政策、经验 Taylor 规则、Riccati reference、linear policy search transfer 以及 benchmark 训练得到的 RL 规则放入 `SVAR` 经验环境中进行历史反事实与长期随机福利比较。虽然这一步仍属于 transfer-based 近似，而不是在经验环境中重新直接训练，但它已经把理论 benchmark 与经验政策评价连接起来，是中期阶段很重要的推进。

## 二、当前最重要的结果与观察

最重要的结果不是“某个 RL 算法已经全面优于传统规则”，而是：项目已经识别出明显的环境依赖性，并且这一结论来自完整矩阵而不是单次幸运训练。

第一，在 benchmark 环境中，理论 Riccati 与 linear policy search 仍然是最强基线。按 `phase7_matrix_summary` 的完整口径，benchmark 下 Riccati 参考规则损失约为 `14.88`，linear policy search 约为 `15.05`；在 RL 算法内部，当前多 seed 固定预算下 `SAC` 最好，均值约为 `16.66`，`TD3` 次之，`PPO` 再后。这说明 benchmark 已经足够严格，当前最稳妥的表述应是：RL 已能学到有意义的政策规则，但尚未在理论可解 benchmark 中取代强线性参考规则。

第二，在 `phase7` 的 `10` 个环境里，若只比较 RL 算法，`SAC` 在 `8` 个环境中胜出，`TD3` 在 `2` 个更强的 ZLB/ELB-tightness 环境中胜出，`PPO` 没有成为任何一个环境的最佳 RL 算法。这一模式很稳定：在 nonlinear 和 asymmetric 环境中，`SAC` 表现最好；在 tighter bound 环境中，`TD3` 更占优势；`PPO` 则随着扭曲增强而更明显地掉队。由此可以支持的中期结论是：算法适配性明显依赖环境类型，off-policy 连续控制算法在约束更强的政策问题中更有优势。

第三，若把外部参考规则也放入同一张表比较，当前最值得注意的事实是：在 `phase7` 的大多数环境里，Riccati reference 或 linear policy search 依然排在 RL 前面，尤其是在 nonlinear 与 asymmetric 环境中更为明显；在 ZLB 三档环境中，它们也仍然强于现有 RL 结果。这并不意味着 RL 没有价值，而是意味着当前引入的非线性、约束和偏好扭曲，虽然已足以改变不同 RL 算法之间的排序，但还不足以彻底推翻 benchmark 线性反馈结构的主导地位。这个判断对正文写作很关键，因为它比“RL 在复杂环境中必然更优”更符合目前证据。

第四，经验环境的第一轮结果给出了一个重要提醒：benchmark 里学到的规则并不能无损迁移到经验环境。`phase8` 的历史福利比较显示，当前 transfer-based 比较中，历史实际政策与经验 Taylor 规则都优于 benchmark 迁移来的 RL 规则；在迁移规则内部，`SAC` 仍优于 `TD3` 和 `PPO`，而 `PPO` 最弱。这说明项目后续最有价值的推进方向不是继续堆 benchmark 训练，而是在经验环境中做 direct-trained policy comparison，并在写作中明确区分 transfer-based 结果与 direct-trained 结果，同时写清 Lucas critique 所带来的方法边界。

第五，从已保存的线性 surrogate 和近似系数看，较优的 RL 规则仍然保留了明显的 Taylor 型结构：对通胀缺口、产出缺口一般保持正响应，并保留一定程度的利率平滑；不同算法的差别更多体现在响应强度和在约束附近的局部形状，而不是完全颠覆可解释的反馈形式。这意味着后续写作完全可以把“可解释性”作为项目优势之一，而不必把 RL 处理成纯黑箱。

## 三、现阶段可以支持的结论

现阶段最扎实的结论有三点。其一，项目已经完成从理论 benchmark 到扩展环境再到经验反事实入口的主线搭建，核心实验框架是完整的。其二，RL 的作用更适合表述为“复杂货币政策控制问题的数值求解工具”，而不是“必然优于传统政策规则的替代者”；至少在当前证据下，强线性参考规则仍然很有竞争力。其三，真正值得写进后续论文主结果的，不是早期单次 tuned PPO 的漂亮个例，而是完整矩阵揭示出来的环境依赖性：`SAC` 在多数扭曲环境中更稳，`TD3` 在 tighter lower-bound 环境中更强，`PPO` 不是通用最优算法。

相应地，现阶段不宜写得过满的结论也有三点：一是不能写“RL 已经全面优于理论最优或强线性基线”；二是不能把 transfer-based 经验结果当成对现实政策的直接规范结论；三是不能把当前 nonlinear 结果解释成“线性规则在非线性环境中已经失效”。这些说法都超出了目前证据支持的范围。

## 四、下一阶段工作重点

从中期之后的主线看，最重要的任务不是继续扩大无差别 case 数量，而是把现有结果组织成可以写论文的证据链。具体而言，应优先推进三件事：第一，围绕 `phase7` 与 `phase8` 主结果整理图表与可视化解释，特别是规则结构、环境异质性和历史反事实路径；第二，在经验环境中补 direct-trained 比较，减少仅靠 transfer 的解释压力；第三，把“理论 benchmark—扩展环境—经验反事实—方法边界”写成一致叙事。若按开题报告的范围控制，中期报告之后不应再把大量精力放在与主线关系较弱的零散补跑上。

综上，项目当前已经不再停留在“RL 能否用于货币政策”的概念验证阶段，而是进入“哪些规则在什么环境下有效、哪些结论可以被稳健写入论文”的收束阶段。这意味着后续工作的关键不只是继续计算，更是把已有结果整理为清晰、克制且有经济学解释力的论文证据。
